# CortaFila

1. Introdução
O Corta Fila é um sistema de agendamento online voltado para barbearias, salões de beleza, esteticistas, manicures, designers de sobrancelhas e serviços similares. A plataforma permitirá que clientes marquem horários de forma simples e rápida, enquanto os estabelecimentos poderão gerenciar sua agenda digitalmente.
________________________________________
2. Objetivo
Facilitar e modernizar o processo de agendamento, reduzindo faltas, otimizando o controle de horários e oferecendo uma experiência moderna tanto para o cliente final quanto para o estabelecimento.
________________________________________
3. Público-alvo
•	Barbearias
•	Salões de beleza
•	Esteticistas
•	Manicures e pedicures
•	Designers de sobrancelhas
•	Negócios de beleza e estética
________________________________________
4. Tecnologias utilizadas
•	Backend: Java Spring Boot
•	Frontend: React.js
•	Banco de dados: MySQL
•	Integração de mensagens: API do WhatsApp 
•	Hospedagem: Google Cloud
